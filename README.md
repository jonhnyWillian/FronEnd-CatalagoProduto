# Catálogo de Produtos

Projeto desenvolvido com **Next.js, React e TypeScript** que consome a API pública **DummyJSON** para exibir um catálogo de produtos.

## Pré-requisitos

Antes de iniciar, é necessário ter instalado:

* Node.js (versão 18 ou superior)
* npm ou yarn

## Como instalar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/catalogo-produtos.git
```

2. Acesse a pasta do projeto:

```bash
cd catalogo-produtos
```

3. Instale as dependências:

```bash
npm install
```

ou

```bash
yarn install
```

## Como rodar o projeto

Execute o comando abaixo para iniciar o servidor de desenvolvimento:

```bash
npm run dev
```

Depois, abra no navegador:

```
http://localhost:3000
```

A aplicação estará disponível localmente.
